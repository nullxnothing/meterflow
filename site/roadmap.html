Product Map

Agent payment rails.
Meterflow control plane.

x402 and the Machine Payments Protocol (MPP) make machine-payable API calls possible. Meterflow adds the operating layer around them: route pricing, wallet policy, spend limits, Solana USDC settlement, receipts, provider revenue, discovery, and MFLOW utility.
